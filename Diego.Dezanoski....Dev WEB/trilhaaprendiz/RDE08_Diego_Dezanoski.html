Tag<sub>

Tal tag serve para subscrever o texto. O texto aparece meio caractere abaixo da linha normal.

Este texto contem parte subscrita .

Codigo do exemplo usando essa tag :

 < p > Este texto contem parte  < sub >subscrita< /sub >. < /p >

Tag <sup>

Tag usada a fim de fazer o contrario da tag sub, esta traz o texto sobrescrito

Este texto contem parte sobrescrita.

Codigo do exemplo usando essa tag :

 < p > Este texto contem parte  < sup >sobrescrita< /sup >. < /p >

Tag <bdo >

Serve para especificar a direcao do texto, da esquerda para direita. Uma substiuição bidirecional Com o atrivuto dir que apos isso recebera valores:
ltr: Indica que o texto será da Esquerda para a Direita ( L eft- T o- R ight).
ou
rtl: Indica que o texto ficará da Direita para a Esquerda ( R eft- T o- Light )
Exemplo:

Este paragrafo irá da esquerda para a direita.


Este paragrafo irá da direita para a esquerda.

Codigo do exemplo usando essa tag :

 
          < bdo dir="rtl">
          <p>Este paragrafo irá da esquerda para a direita.</p>
          </bdo>
          <br>
    
          <bdo dir="ltr">
          <Este paragrafo irá da direita para a esquerda.</p>
          </bdo>

Tag <code>

é usada para definir um pedaço de código de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

Exemplo:

Texto de inicio (Codigo ou funcao a ser colocada). Restante do texto.

Codigo do exemplo usando essa tag :

 < p > Texto de inicio < code >(Codigo ou funcao a ser colocada)</code>. Restante do texto.</p>

Tag <pre>

Serve para marcarção de um texto pre formatado, O texto nessa tag é exibido em uma fonte de largura fixa e o texto preserva espaços e quebras de linha.
Exemplo a abaixo:
